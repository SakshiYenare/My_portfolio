Personal AI Chatbot using ESP32

Developed a voice-interactive AI chatbot using the ESP32 microcontroller, enabling real-time communication through speech and text. The system captures user voice input via a microphone, converts it to text using Google Speech API, and sends it to cloud-based AI platforms like OpenAI or Dialogflow. The chatbot then responds with intelligent replies, which are delivered through both an OLED display and a speaker using text-to-speech (TTS) synthesis. This low-cost, portable assistant is ideal for personal use, home automation, and educational applications. The project demonstrated seamless integration of hardware and software, cloud API communication, and real-time interaction, all built on a compact, energy-efficient platform.
